# SQLite Viewer - 数据查看器

这是一个使用纯前端实现的 SQLite 数据库查看器。

## 使用方法

在 VSCode 中：

1. 安装 **SQLite Viewer** 扩展 (mtachter)
2. 按 `Ctrl+P` 打开命令面板
3. 输入 `Open Database` 或 `SQLite: Open Database`
4. 选择 `backend/game_guide.db` 文件

## 或者使用在线查看

如果你无法安装扩展，可以：

1. 打开 https://sqliteviewer.app/
2. 将 `backend/game_guide.db` 文件拖入浏览器
3. 直接查看所有表数据

## 数据库路径

```
d:\线上实战\game-guide-platform\backend\game_guide.db
```
